### Getting started

```bash
# install all dependencies
$ npm install

# start presentation & have fun
$ npm run start
```

### Build

```bash
# build a standalone version of presentation into dist folder
$ npm run build
```

### Deploy
```bash
# build the standalone version and deploys it to github pages
$ npm run deploy
```
